# CPU Simulator

Basic CPU Simulator built in c++.

## Third Party:
raysan5/aylib for rendering.
nlohmann/json for save/load. (to be implemented)

## Building:
In the root diretory:
```
touch build
cd build
cmake ..
make
```
Works on MacOS by default.
Windows need some tweaks but kinda works.
